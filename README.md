Welcome to the source code of my personal portfolio website! This is an interactive, responsive 3D web experience built to showcase my skills, projects, and journey as a Frontend Developer and AI Enthusiast.

## ✨ Features
- **3D Interactive Elements**: Integrated using Three.js to provide an engaging background and floating elements.
- **Modern UI/UX**: Clean, pixel-perfect, and minimal aesthetic focusing on readability and presentation.
- **Fully Responsive**: Adapts seamlessly to all devices (desktop, tablet, mobile).
- **Projects Showcase**: Highlights the recent things I've built, including a Weather App, Vedam Clubs site, and a Task Filter App.
- **Dynamic Skill Orbits**: A creative way to display my technology stack.

## 🛠️ Tech Stack
This project was built entirely with core web technologies for maximum performance and explicit control over styling:
- **HTML5**: Semantic tags for structured content.
- **CSS3 / Vanilla CSS**: Advanced styling, custom animations, variables, and responsive grids/flexbox without external CSS libraries.
- **JavaScript (ES6+)**: Custom DOM manipulation, interactive logic, and event handling.
- **Three.js**: Utilized for 3D canvas rendering and interactive visual effects.

## 📁 Project Structure
- `index.html` - The main structure of the webpage and content elements.
- `style.css` - comprehensive styling, animations, colors, and layout rules.
- `main.js` - Logic for Three.js animations, smooth scrolling, interaction tracking, and dynamic element rendering.

## 🚀 Getting Started Locally
To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/praptibhandari/portfolio-website.git
   ```
2. Open the directory:
   ```bash
   cd portfolio-website
   ```
3. Since this is a static site with no build steps, you can just open the `index.html` file right in your browser, or use an extension like VS Code Live Server.
